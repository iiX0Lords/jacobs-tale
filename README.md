## How to Build

Successful builds are stored inside of projectroot/dist

1.  **Clone this repository:**
    ```bash
    git clone -b luau https://github.com/iiX0Lords/jacobs-tale.git
    ```
2.  **Run inside of project root:**
    ```bash
    "love2d/kaledis/kaledis.exe" compile
    ```
